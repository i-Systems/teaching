Problem 2¶

In this problem, we are going to perform anomaly detection using LSTM layers

  • The dataset given below is composed of data from four bearing sensors on a single bearing. In other words, there were four sensors attached to the bearing. We have collected the data from 2004-02-12 to 2004-02-19. We noticed that some time after 2004-02-15, the bearing started to malfunction. The main objective of this problem is to find exact date and time when the bearing started to malfunction. We will use the concept of reconstruction error discussed in the class.

(0) Import libraries and load the data which you can download it from this link

  • We have solid belief that the bearing did not malfunction until 2004-02-15 12:52:39
  • Set data collected after 2004-02-15 12:52:39 as test dataset to find the exact date and time when the malfunction (anomaly) happened
  • Use data collected before 2004-02-15 12:52:39 as train dataset to perform anomaly detection
