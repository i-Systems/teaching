Problem 1. Pretext Tasks¶

(1) Prof. Lee has explained about various pretext tasks you can use for the self-supervised learning. Choose one pretext task and explain the mechanism of that pretext task for self-supervised learning in detail. (Exclude the Rotation Network)

(2) Draw the model (network) used in the pretext task chosen at (1) and explain in detail (Specify the function of each part of the model)

  • You may use figures in articles

(3) Explain downstream tasks that the pretext task specified at (1) can be applied

Answer

(1) Context Prediction is a pretext task that creates nine 3x3-shaped patches from an input image and then finds out the relative position of an arbitrary patch based on the patch located in the middle. In Context Prediction, the input is a center patch and one non-center patch, and the output is a value representing the relative position of the non-center patch from 1 to 8. Context Prediction is a type of Self-supervised Learning because a patch can be created from an input image and a label can be automatically generated even if a person does not directly create a label by selecting a random patch.

(2) Since the input of Context Prediction is two patches, there are two branches in the network that receive each of two patches. Two branches share the same weights and there is a final layer that aggregates the results of two branches and use them to make predictions.

(3) The model trained through the pretext task described above can obtain features of images very well. After fixing the weights of the model, downstream tasks like classification can be performed based on the obtain features.

Problem 2. Self-supervised Learning (RotNet) vs. Transfer Learning¶

  • Self-supervised Learning (SSL) is very similar to Transfer Learning but they are different in the point that Self-supervised Learning uses unlabeled data but Transfer Learning requires labeled data
  • In the real world, there are more unlabeled data than labeled ones.
  • We are going to compare the performance between the Self-supervised Learning and Transfer Learning

n_samples = X_train.shape[0]
X_rotate = np.zeros(shape = (n_samples * 4,
                             X_train.shape[1],
                             X_train.shape[2]))
Y_rotate = np.zeros(shape = (n_samples * 4, 4))

for i in range(n_samples):    
    img = X_train[i]
    X_rotate[4 * i - 4] = img
    Y_rotate[4 * i - 4] = tf.one_hot([0], depth = 4)
    
    # 90 degrees rotation
    X_rotate[4 * i - 3] = np.rot90(img, k = 1)
    Y_rotate[4 * i - 3] = tf.one_hot([1], depth = 4)
    
    # 180 degrees rotation
    X_rotate[4 * i - 2] = np.rot90(img, k = 2)
    Y_rotate[4 * i - 2] = tf.one_hot([2], depth = 4) 
    
    # 270 degrees rotation
    X_rotate[4 * i - 1] = np.rot90(img, k = 3)
    Y_rotate[4 * i - 1] = tf.one_hot([3], depth = 4)
