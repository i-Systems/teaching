Problem 1: Gradient Descent¶

In this problem, we are going to solve the regression problem with batch and stochastic gradient descent methods. The objective of this problem is to compare the loss of each method. Since this problem is simple, the loss would reduce rapidly. So, iterate no more than 100 times.

# Solution
LR = 0.1
n_iter = 100
m = 100

# initialization
theta = np.random.randn(2,1)
loss = []
for i in range(n_iter):
    g = 2/m * X.T.dot(X.dot(theta)-y)
    theta = theta - LR*g
    l = 1/m*np.sum((X.dot(theta)-y)**2)
    loss.append(l)

plt.figure()
plt.plot(loss)
plt.title('batch gradient descent')
plt.show()

(2) Find the fitted line with the stochastic gradient descent method and plot the loss. We use a stochastic gradient descent method to prevent the model converging to a local minimum, but the stochastic gradient descent method makes it hard to converge to a global minimum, too. To solve this problem, we are going to reduce the learning rate gradually. In other words, use a large learning rate at first and decrease it gradually.

# Solution
n_epochs = 1
t0, t1 = 5, 50

def LearningRate(t):
    return t0/(t+t1)

# initialization
theta = np.random.randn(2,1)

loss = []
for epoch in range(n_epochs):
    for i in range(m):
        idx = np.random.randint(m)
        xi = X[idx:idx+1]
        yi = y[idx:idx+1]
        g = xi.T.dot(xi.dot(theta)-yi)
        LR = LearningRate(epoch*m+i)
        theta = theta - LR*g
        l = 1/m*np.sum((X.dot(theta)-y)**2)
        loss.append(l)

plt.figure()
plt.plot(loss)
plt.title('Stochastic gradient descent')

plt.show()

Problem 2: Regularization¶

The regularized least-squares problem has the form

$$ \min_{\theta} \;\lVert A\theta -y\rVert_2^2 + \lambda \lVert \theta \rVert_2^2$$


(a) Show that the solution is given by

$$ \hat{\theta} = \left( A^T A + \lambda I_n \right)^{-1} A^T y $$
  • Do not use the method of Lagrangian multipliers

(sol a) Sum of 2-norms of multiple vectors can be expressed as a 2-norm of a single vector.


$$ \begin{align*} \left\Vert A\theta - y\big\Vert^2_{2} + \big\Vert \sqrt{\lambda}I\theta-0\right\Vert^2_{2} & = \Bigg\Vert \begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix} \theta- \begin{bmatrix} y\\ 0 \end{bmatrix} \Bigg\Vert^2_{2} \end{align*}$$


$$ \hat{\theta} = \left( \begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix}^T \begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix}\right)^{-1}\begin{bmatrix} A\\ \sqrt{\lambda}I \end{bmatrix}^T \begin{bmatrix} y\\ 0 \end{bmatrix} = \left( A^T A + \lambda I_n \right)^{-1} A^T y $$



(b) Write down a gradient descent algorithm for the given optimization problem.

  • Hint: Note that $$ \;\lVert A\theta -y\rVert_2^2 = (A\theta - y)^T(A\theta - y)$$
    Then, you can differentiate the above equation to compute the gradient. Likewise, you can compute the gradient of the regularizer.

(sol b) Let $J(\theta) = \Vert A\theta - y \Vert_2^2 + \lambda \Vert \theta \Vert_2^2$. A gradient descent algorithm is formulated as the following,


$$ \theta \leftarrow \theta - \eta \frac{\partial{J}}{\partial{\theta}}. $$

Need to compute $\frac{\partial{J}}{\partial{\theta}}$.


$$ \begin{align*} g_{\text{projection}} &= \frac{\partial}{\partial{\theta}} \Vert A\theta - y \Vert_2^2 \\ &= \frac{\partial}{\partial{\theta}} (A\theta - y)^T(A\theta - y) \\ &= \frac{\partial}{\partial{\theta}} (\theta^T A^T A \theta - 2\theta^T A^T y - y^Ty) \\ &= 2A^TA \theta - 2A^Ty \\ \\ g_{\text{regularizer}} &= \frac{\partial}{\partial{\theta}} \Vert \theta \Vert_2^2 \\ &= \frac{\partial}{\partial{\theta}} \theta^T \theta \\ &= 2 \theta. \end{align*} $$

Therefore,


$$ \begin{align*} \frac{\partial{J}}{\partial{\theta}} &= g_{\text{projection}} + \lambda g_{\text{regularizer}} \\ &= 2A^TA \theta - 2A^Ty + \lambda 2 \theta. \end{align*} $$

Then, the gradient descent algorithm is formulated as the following


$$ \theta \leftarrow \theta - \eta(2A^TA \theta - 2A^Ty + \lambda 2 \theta). $$

(c) Based on the result of (b), describe the role of regularizer term.

  • Hint: Gradient $g$ is computed by $ g = g_{\text{projection}} + g_{\text{regularizer}} $.



(sol c)

$g_{\text{regularizer}}$ tends to make $\theta$ converge to zero. Since $g = g_{\text{projection}} + \lambda g_{\text{regularizer}}$, we can see that the regularizer $g_{\text{regularizer}}$ tries to keep the value of $\theta$ small.

(d) Describe results of (a) and (b) have the same meaning.



(sol d)

Likewise, (a) shows that

$$ (A^T A)^{-1} A^Ty \quad \rightarrow \quad (A^T A + \lambda I_n)^{-1} A^Ty. $$

From the solution of (a),

$$ \nabla_\theta^2 f(\theta) = 2\left(A^TA + \lambda I_n \right)^T \geq 0 \quad \left(\because A^TA = \lVert A \rVert^2,\quad \lambda \geq 0 \right)$$

so, this problem is a convex problem

From solution of (b), if the number of iteration become enough, $\theta_{i+1} \approx \theta_i = \hat{\theta} $ in equation $\theta_{i+1} \leftarrow \theta_i - \alpha g\left(\theta_i \right) $, so


$$ \begin{align*} g\left(\hat{\theta} \right) = 2A^TA \theta - 2A^Ty + \lambda 2 \theta = 0\\ \end{align*} $$

$$ \begin{align*} \left( A^T A + \lambda I_n \right)\hat{\theta} = A^T y \end{align*} $$$$ \begin{align*} \therefore \hat{\theta} = \left( A^T A + \lambda I_n \right)^{-1} A^T y \end{align*} $$


Therefore (a) and (b) are the same meaning

import warnings

# Solution (Tensorflow 1)
import tensorflow as tf
from sklearn.preprocessing import MaxAbsScaler
import tensorflow.compat.v1 as tf
tf.disable_v2_behavior()


LR = 0.4                                                   
n_iter = 10000

train_x = x.reshape(-1, 1)
train_y = y.reshape(-1, 1)

def fitting(d):
    train_X = np.hstack([train_x**(i+1) for i in range(d)])
    train_X = MaxAbsScaler().fit_transform(train_X)
    train_X = np.asmatrix(train_X)

    x = tf.placeholder(tf.float32, [n, d])
    y = tf.placeholder(tf.float32, [n, 1])

    w = tf.Variable(tf.random_normal([d, 1]))
    b = tf.Variable(tf.random_normal([1, 1]))

    y_pred = tf.add(tf.matmul(x, w), b)
    loss = tf.square(y_pred - y)
    loss = tf.reduce_mean(loss)

    optm = tf.train.AdamOptimizer(LR).minimize(loss)
    init = tf.global_variables_initializer()

    loss_record = []
    with tf.Session() as sess:
        sess.run(init)
        for epoch in range(n_iter):                                                                         
            sess.run(optm, feed_dict = {x: train_X, y: train_y})  
            loss_record.append(sess.run(loss, feed_dict = {x: train_X, y: train_y}))

        w_val = sess.run(w)
        b_val = sess.run(b)

    Xp = np.hstack([xp**(i+1) for i in range(d)])
    Xp = MaxAbsScaler().fit_transform(Xp)
    Xp = np.asmatrix(Xp)

    yp = Xp*w_val + b_val
    return yp

xp = np.linspace(0, 4, 100).reshape(-1,1)
plt.figure(figsize=(16, 6))
plt.subplot(121)
plt.plot(train_x, train_y, 'ko')
plt.plot(xp, fitting(3), 'r')
plt.title('Data', fontsize = 15)
plt.xlabel('X', fontsize = 15)
plt.ylabel('Y', fontsize = 15)
plt.grid(alpha = 0.3)
plt.subplot(122)
plt.plot(train_x, train_y, 'ko')

# Solution (Tensorflow 2)
import tensorflow as tf
from sklearn.preprocessing import MaxAbsScaler


LR = 0.4                                                   
n_iter = 10000

train_x = x.reshape(-1, 1)
train_y = y.reshape(-1, 1)

def fitting(d):
    train_X = np.hstack([train_x**(i+1) for i in range(d)])
    train_X = MaxAbsScaler().fit_transform(train_X)
    train_X = np.asmatrix(train_X)

    w = tf.Variable(np.zeros([d, 1]))

    optm = tf.keras.optimizers.Adam(learning_rate = LR)

    for _ in range(n_iter):
        with tf.GradientTape() as tape:
            tape.watch(w)
            cost = tf.reduce_mean(tf.math.square(tf.matmul(train_X, w) - train_y))
        gradients = tape.gradient(cost, [w])
        optm.apply_gradients(zip(gradients, [w]))

    Xp = np.hstack([xp**(i+1) for i in range(d)])
    Xp = MaxAbsScaler().fit_transform(Xp)
    Xp = np.asmatrix(Xp)

    yp = tf.matmul(Xp, w)
    return yp

xp = np.linspace(0, 4, 100).reshape(-1,1)
plt.figure(figsize=(16, 6))
plt.subplot(121)
plt.plot(train_x, train_y, 'ko')

plt.title('Data', fontsize = 15)
