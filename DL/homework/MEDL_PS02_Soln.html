Problem 3: Understanding Probability in Logistic Regression

In this problem, we are going to understand probability in the logistic regression.

Note: you need to write down full codes for this problem by yourself. Do not just use sklearn library.

Linear Regression vs. Logistic Regression

In the case of categorical (0 or 1) data, linear regression is not possible to properly represent the relationship between input and output data. As shown in the figure on the right, logistic regression adds a non-linear characteristic (sigmoid function) to linear regression and fits the relationship between input and output data into a curve.

We can define the S-Curve as a probability by the characteristics of the sigmoid function.

Step 2. Logistic Regression with Gradient Descent

(1) Define the sigmoid (logistic) function.


$$h_{\omega}(x) = h(x\,;\omega) = \sigma \left(\omega^T x\right) = \frac{1}{1+e^{-\omega^T x}}$$
In[13]:
# Your code here

def h(x,w):
    return 1/(1 + np.exp(-x*w))

(2) [hand written] Write the gradient of log likelihood.

Solution

The likelihood function is


$$ \begin{align*} L(\omega) &= \prod\limits_{i=1}^{m}P(y^{(i)}|x^{(i)};\omega)= \prod\limits_{i=1}^{m}(h_\omega (x^{(i)})^{y^{(i)}})(1-h_\omega (x^{(i)}))^{(1-y^{(i)})}\\\\ \ell(\omega) &= \frac{1}{m} \log L(\omega) = \frac{1}{m}\sum\limits_{i=1}^m y^{(i)} \{\log(h_\omega (x^{(i)})) + (1-y^{(i)}) \log(1-h_\omega(x^{(i)}))\}\\\\ \nabla \ell(\omega) &= \frac{1}{m} \sum\limits_{i=1}^{m} \left\{\frac{y^{(i)}}{h_\omega (x^{(i)})} \frac{\partial h_\omega}{\partial \omega} -\frac{1-y^{(i)}}{1-h_\omega (x^{(i)})} \frac{\partial h_\omega}{\partial \omega}\right\}\\\\ \frac{\partial h_\omega}{\partial \omega} &= \frac{\partial \sigma(z^{(i)})}{\partial z^{(i)}} \frac{\partial z^{(i)}}{\partial \omega}= \frac{\partial \sigma(z^{(i)})}{\partial z^{(i)}}x^{(i)} \\\\ \nabla \ell(\omega) &= \frac{1}{m} \sum\limits_{i=1}^{m} (y^{(i)}-h_\omega (x^{(i)}))x^{(i)} \end{align*} $$


Thus,


$$\nabla \ell(\omega) = \begin{bmatrix} \frac{\partial \ell(\omega)}{\partial \omega_0} \\ \frac{\partial \ell(\omega)}{\partial \omega_1} \\ \frac{\partial \ell(\omega)}{\partial \omega_2} \end{bmatrix} = X^T \left(y-h_{\omega}(x)\right) = X^T \left(y-\sigma(X \omega)\right) $$


(3) [hand written] Write the gradient descent algorithm for this problem.

Note: this problem is a maximization problem.

Solution
This problem is a maximization problem, so we need to make it minimization problem to use gradient descent method.

Since $\min\{\nabla \ell(\omega)\} = \max\{-\nabla \ell(\omega)\}$, we can convert this problem to minimization problem by simply change the sign of gradient.


$$\omega \leftarrow \omega - \eta \left( - \nabla \ell(\omega)\right)$$


(4) Initialize $\omega$ and update it by the gradient descent method.

Step 3. Linear Regression with Gradient Descent

(1) [hand writtern] The below is the objective function of linear regression.


$$ \min_{\theta} ~ \lVert \hat y - y \rVert_2^2 = \min\limits_\theta {\lVert X\theta - y \rVert}_2^2$$

Write the gradient of least squares solution.

Solution


let $J(x) = \lVert X\theta-y\rVert^2_{2}$

$$ \begin{align*} J(x) &= (X\theta-y)^T(X\theta-y)\\\\ &= (\theta^TX^T-y^T)(X\theta-y)\\\\ &= \theta^TX^TX\theta - \theta^TX^Ty - y^TX\theta+y^Ty\\\\ \end{align*} $$

Thus,

$$ \begin{align*} \frac{\partial{J}}{\partial{\theta}} &= X^TX\theta+(X^TX)^T\theta - X^Ty - {(y^TX)}^T\\\\ &= 2(X^TX\theta - X^Ty) \end{align*} $$

(2) Initialize $\theta$ and update it by the gradient descent method.

Note: this problem is a minimization problem.

Step 5. Cost Function of Logistic Regression

The cost function summarizes how well the model is behaving. In other words, we use the cost function to measure how close the models predictions are to the actual outputs.

Log likelihood


$$\ell(\omega) = \frac{1}{m} \log L(\omega) = \frac{1}{m}\sum\limits_{i=1}^m y^{(i)} \{\log(h_\omega (x^{(i)})) + (1-y^{(i)}) \log(1-h_\omega(x^{(i)}))\} $$


Thus, we can define the cost for two cases separately:


$$ \text{cost}(h_{\theta}(x),y) = \begin{cases} -\log (h_{\theta}(x)) \quad &\text{if } y = 1\\ -\log (1-h_{\theta}(x)) \quad &\text{if } y = 0 \end{cases} $$
